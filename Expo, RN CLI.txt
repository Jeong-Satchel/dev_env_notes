[Expo]
- 모든 불편한 작업을 대신 해줌. 더 다양한 기능을 제공해줌.
- app build 작업을 대신 해줌. 
- 장점: easy setup. Expo 앱을 통해 스마트폰에서 앱 테스트 기능 지원.
        버그 감소. React Native 관련 업데이트 자동 실행. more modules. API 등.
        Windows에서도 iOS 앱 build 과정 대신해줌. 
- 단점: native 파일 접근 불가. 제어 불가. 

[React Native CLI]
- 장점: native 파일들에 접근 가능. Xcode 파일 & Android Studio 파일에 직접 접근 가능.
        기업 수준에서는 필요. 개인 개발자 수준에서는 굳이 제어할 필요가 없음. 
- 단점: more time to configure. 앱 테스트하려면 Xcode 계정 등 별도의 작업 더 필요함.
        버그를 엄청 많이 경험하게 됨. less modules, API.
        Mac이어야만 iOS 앱 build 가능.
