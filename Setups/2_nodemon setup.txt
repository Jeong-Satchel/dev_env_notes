npm i nodemon -g
: global로 설치해놓으면 프로젝트별로 매번 새로 설치할 필요가 없어짐.

===============================================================
npm init

package.json
{
  "main": "index.js",         // 디폴트로 주시할 파일
  "scripts": {
    "start": "nodemon"
  },

npm start 
  1) 'node index.js'를 실행시키고,
  2) index.js의 js코드에 문제가 있는지 지속적으로 확인.
